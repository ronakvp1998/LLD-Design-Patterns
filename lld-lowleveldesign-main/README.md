# LLD Low-Level Design Patterns

## Repository Overview

This repository contains Java examples for design patterns, SOLID principle demonstrations, and interview-style low-level design (LLD) problem solutions.

Each major topic is listed below with a direct link to its folder so you can quickly jump to the code. The LLD problem set section is kept separate for clarity.

## How to Use

1. Click the folder links above to open the topic or problem set you want to explore.
2. Use your IDE or GitHub file browser to view Java implementations under `src/main/java/`.
3. Add new folders and update this README to keep the topic index extensible.

## Contributing

- Add a new folder under the relevant package when introducing a new pattern or problem.
- Keep LLD problem examples separate from the design pattern sections.
- Add or update table rows in this README so the repository index stays current.
